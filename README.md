# APP DE FOTOGRAFÍAS EN LINEA FOTAZA

Es una aplicación web que permite almacenar, ordenar, buscar, vender y
compartir fotografías en línea, a través de Internet. 

## GUIA DE USO

### LEVANTAR PROYECTO
- Instale las dependencias con `npm install`

- Use `npm run dev` para iniciar en modo watch y poder editar y recargar el servidor
